 File Organizer

 Project Overview

This project is a Python File Organizer that reads all files from a folder, creates folders based on file extensions, moves files into their respective folders, and generates a report.

 Features

- Read files from a folder
- Create folders automatically
- Organize files by extension
- Generate report.txt
- Display total files processed
- Display files in each category
- Display moved files
- Save timestamp of execution

 How to Run

1. Place files inside the `sample_files` folder.
2. Run:
python main.py
